# Maincrafts Technology Internship — Task 1: Data Analytics & BI

## Overview
This project involves cleaning, analyzing, and visualizing the Superstore sales dataset 
using SQL and Power BI to extract actionable business insights.

## Deliverables
- `cleaned_sales_data.xlsx` — Cleaned Superstore dataset
- `queries.sql` — SQL queries (Region-wise, Category-wise, Monthly, Discount impact)
- `TASK 1.pbix` — Power BI dashboard
- `Screenshot 2026-06-20 045438.png` — Dashboard preview
- `Business_Insight_Report_Arjit.pdf` — Final business insight report

## Key Findings
- West region generated the highest sales ($725,457)
- Technology was the most profitable category ($145,454)
- Higher discounts correlate with lower/negative profit
- November and December were peak sales months

SQL, Power BI, Excel
